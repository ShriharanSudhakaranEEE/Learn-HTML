Types of links

There are three different types of links are present in HTMl

  1. Absolute Link
  2. Relative Link
  3. Internal Link

Absolute Link

An absolute link is a type of link which
is used to link the webpage with the another
webpage that is to redirect to the another webpage.
Ex: Linking youtube link in the webpage
Google

Relative Link

A relative link is a type of link which
is used to link the local file
(i.e) linking the html file with css file or anyother
local file.
Ex: Linking html with css

Intenal Link

An interal link is a type of link which
is used to link the different section of
the webpage. Ex: Linking section
