List and its types

There are three different types of List

  1. Ordered list
  2. Unordered list
  3. Descriptive list

Ordered List

Ordered list is a list used to create in numerical form such as 1,2,3,etc

  1. The syntax is ol for ordered list
  2. The li is used as syntax for creating list

Unordered List

Unordered list is a list used to create list in bulletin form

  • The syntax of the unorderd list is ul
  • The syntax of the list is li

Descriptive List

Descriptive list is a list used to create list in question and answer types

The syntax of descriptive list
It has two part dt -- descriptive term and
Another part is dd -- descriptive details.
